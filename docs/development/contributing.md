# Contributing

Thank you for your interest in contributing to Neurolight! Please see the [CONTRIBUTING.md](https://github.com/Neuro-Light/neurolight-prototype/blob/main/CONTRIBUTING.md) file in the repository for detailed guidelines.

## Quick Start

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests: `uv run pytest tests/`
5. Run linting: `uv run ruff check .`
6. Submit a pull request

## Code Quality

We use:
- **Ruff** for linting and formatting
- **MyPy** for type checking
- **Pytest** for testing

Make sure all checks pass before submitting your PR.
